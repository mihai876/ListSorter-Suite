ListSorter Suite — Сортировщик списков на 7 языках
ListSorter Suite — коллекция из семи независимых реализаций универсального сортировщика списков с расширенными возможностями. Каждая версия работает на своём языке программирования и предлагает множество алгоритмов сортировки, визуализацию процесса, замер производительности и экспорт результатов.

✨ Уникальные возможности (общие для всех версий)
📋 Поддержка различных типов данных: числа, строки, даты.

🔢 Множество алгоритмов сортировки:

Пузырьковая (Bubble Sort)

Быстрая (Quick Sort)

Сортировка слиянием (Merge Sort)

Встроенная (стандартная) сортировка языка

Сортировка выбором (Selection Sort)

Сортировка вставками (Insertion Sort)

📊 Визуализация процесса (ASCII-графика в CLI, анимация в веб).

⏱️ Замер времени выполнения каждого алгоритма.

📈 Сравнительный анализ производительности алгоритмов.

💾 Экспорт результатов в CSV и JSON.

🔄 Генерация случайных списков для тестирования.

🎯 Интерактивный режим с выбором алгоритма и параметров.

🖥️ Интерфейсы:

Командная строка (CLI) с богатыми опциями.

Веб-интерфейс в PHP и JavaScript.
